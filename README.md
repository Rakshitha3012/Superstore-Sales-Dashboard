# Superstore Sales Dashboard

## Objective
To create an interactive Power BI dashboard that visualizes Superstore sales data and provides meaningful business insights.

## Dataset
- Superstore Sales Dataset (train.csv)

## Tools Used
- Microsoft Power BI

## Dashboard Features
- Total Sales Card
- Sales by Category
- Sales by Sub-Category
- Sales Trend by Order Date
- Sales by Segment
- Region Slicer

## Key Insights
- Technology is the highest-selling category.
- Phones and Chairs are among the top-selling sub-categories.
- The Consumer segment contributes the largest share of sales.
- The dashboard allows interactive filtering by region.
